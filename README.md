# Meu Portfólio Pessoal 🚀

Um portfólio Single Page Application desenvolvido como projeto acadêmico para matéria de desenvolvimento web demonstrar minhas habilidades e projetos na área de desenvolvimento de software.

## Objetivo

O principal requisito deste projeto foi construir uma página web moderna, responsiva e funcional utilizando **apenas tecnologias nativas**, sem o auxílio de frameworks ou bibliotecas externas (como Bootstrap, Tailwind ou React), aplicando boas práticas de componentização e estilização manual.

## Funcionalidades

- **Single Page Application:** Navegação por âncoras com *scroll* suave.
- **Design Responsivo:** Layout adaptável para dispositivos móveis e desktop utilizando Flexbox e CSS Grid.
- **Dark Mode:** Alternância de temas (Claro / Escuro) com preferência salva no `localStorage`.
- **Menu Mobile:** Menu estilo hamburguer interativo para telas menores.
- **Active State:** Destaque dinâmico no item do menu correspondente à seção visível durante a rolagem da página.
- **Validação de Formulário:** Validação de campos (nome, e-mail e mensagem) feita inteiramente com Vanilla JavaScript, incluindo feedback visual e modal de sucesso.

## Tecnologias Utilizadas

- **HTML5:** Estrutura semântica.
- **CSS3:** Estilização pura com variáveis globais (`--var`), transições e Media Queries.
- **JavaScript:** Lógica de interface, observer de rolagem e manipulação do DOM.
